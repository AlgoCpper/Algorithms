## 문제 접근
1. 한 문자열을 고정시키고 다른 문자열의 한 문자씩 비교한다.
2. (같은 경우) LCS(Xn, Ym) = ( LCS(Xn-1, Ym-1), 같은 원소 ) 라고 할 수 있다. 즉, 이전 LCS에 + 1 한 값이다.
3. (다른 경우) 두 문자열이 같은 기호로 끝나지 않는다면 LCS(Xn-1, Ym) 혹은 LCS(Xn, Ym-1) 중 큰 값이 LCS(Xn, Ym)이다.

점화식으로 표현하면 다음과 같다.
```java
if i == 0 || j == 0
	DP[i][j] = 0;
else if X[i] == Y[j]
	DP[i-1][j-1] + 1;
else if X[i] != Y[j]
	max(DP[i-1][j], DP[i][j-1]);
```
